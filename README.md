## Setup

1. Fork & Clone
2. Run `npm install` to install dependencies


## Dependencies

* body-parser: ^1.18.3
* ejs: ^2.6.1
* express: ^4.16.3
* express-sanitizer: ^1.0.4
* method-override: ^2.3.10
* mongoose": ^5.1.4
